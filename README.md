# Square Game

A simple interactive browser game built with HTML, CSS, and JavaScript.

## Overview

This project displays four square tiles that react to user interaction:

- **Square 1**: Shows a random number on hover and restores to `1` on mouse leave. Double-clicking changes the square's background color to a random RGB value.
- **Square 2**: Cycles through red, green, and blue background colors on hover, and resets to white on mouse leave.
- **Square 3**: Changes to a random RGB background color on hover and resets to white on mouse leave.
- **Square 4**: Clicking it applies random background colors to squares 1, 2, and 3; moving the mouse out resets them to white.
- A custom red cursor indicator follows mouse movement inside the main game area.

## Files

- `index.html` &mdash; Main HTML structure
- `style.css` &mdash; Basic page and square styling
- `script.js` &mdash; Interactive behavior and event listeners

## How to run

1. Open `index.html` in a web browser.
2. Hover and click the squares to see the interactions.

## Notes

- No build tools or dependencies are required.
- Works in modern browsers that support standard JavaScript and CSS.
